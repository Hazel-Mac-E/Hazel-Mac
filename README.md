## Overview

Hazel Professional Suite is a complete file automation platform that serves power users and professionals with the rule depth, processing capability, and integration breadth needed for automated file management that handles any organizational challenge. It extends beyond simple file sorting into genuine workflow automation, where file arrival triggers processing pipelines that transform, distribute, and archive content automatically. The application has been continuously refined for over fifteen years, making it one of the most mature and reliable automation tools available.

Shell script and AppleScript execution within rules enables custom processing that extends Hazel's capabilities to any operation that can be scripted, from image conversion through PDF processing to API calls triggered by file arrival. Scripts receive the matched file as an argument and can return status codes that determine whether subsequent rule actions execute. This extensibility means that any automated workflow that begins with a file appearing in a folder can be handled by Hazel.

Subfolder processing with recursive rule application extends monitoring beyond the watched folder into its entire subfolder hierarchy, enabling deep organizational structures that maintain themselves without creating separate rules for each subfolder level. Rules can be set to process only immediate children or to recurse to any depth, and subfolder-specific rules override parent folder rules when more specific handling is needed. This capability handles complex folder structures like project hierarchies and media libraries.

Pattern matching with date formatting, text extraction, and regex support enables rules that respond to complex file naming patterns, extracting meaningful data from file names and using it to determine file destination, new name, or processing action. Spotlight metadata conditions match on content-level attributes like image dimensions, document author, audio duration, and camera model. This deep matching capability means that rules can distinguish between files that appear superficially similar but require different handling.

Hazel Professional Suite is the file automation platform for users who believe that repetitive file management is a problem that should be solved permanently through automation rather than addressed repeatedly through manual effort. Its depth of matching conditions, breadth of available actions, and reliability over years of continuous background operation make it the file organization tool that works as hard as you do.

## Performance & Integration

Hazel is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Hazel processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
